# Tools Endpoints

All Croud tools will follow the same structure for the following endpoints - individual tools may expose additional endpoints as required but this "core" schema must be followed in all cases

# Systems

# Register a System with the Tool.

A system represents a consuming entity that maintains its own set of clients. A System and Client uniquely identify a client across all registered systems (Tenants).

  • URL

    /systems

  • Method:

    POST

  • URL Params

    None

  • Data Params

    {
    "id": "08ceaae6-11c6-4326-84b0-80c86c8715ac",
    "name": "Croud Control"
}

  • Success Response:

    • Header: Location:/systems/08ceaae6-11c6-4326-84b0-80c86c8715ac
      Code: 201 CREATED
      Content: { "id": "08ceaae6-11c6-4326-84b0-80c86c8715ac", "name": "Croud Control", "registeredDate": "2018-03-08T14:43:32Z" }

  • Error Response:

    • Code: 400 BAD REQUEST
      Content: { "error" : "system id <system_id> is already registered with" }

    OR

    • Code: 500 SERVER ERROR
      Content: { "error" : "description" }

  • Sample Call:

      $.ajax({
    url: "systems",
    dataType: "json",
    type : "POST",
    data : '{ "id": "08ceaae6-11c6-4326-84b0-80c86c8715ac", "name": "Croud Control" }',
    success : function(r) {
      console.log(r);
    }
  });

# Clients

# Register a Client with the Tool for specified SystemId.

  • URL

    /systems/:systemId/clients

  • Method:

    POST

  • URL Params

    systemId=[UUID]

  • Data Params

    {
    id: "123456",
    name: "Croud Agency"
}

  • Success Response:

    • Header: Location:/08ceaae6-11c6-4326-84b0-80c86c8715ac/clients/123456
      Code: 201 CREATED
      Content: { clientId: "123456", registeredDate: "2017-10-18T14:43:32Z" }

  • Error Response:

    • Code: 400 BAD REQUEST
      Content: { error : "id <client_id> is already registered with service under system <system_id>" }

    OR

    • Code: 500 SERVER ERROR
      Content: { error : "description" }

  • Sample Call:

      $.ajax({
    url: "/08ceaae6-11c6-4326-84b0-80c86c8715ac/clients",
    dataType: "json",
    type : "POST",
    data : '{ "id": "123456", "name": "Croud Agency" }',
    success : function(r) {
      console.log(r);
    }
  });
