# Tools

The tools platform consists of a number of components each of which falls into two main categories:

  1. Framework

    These components are a series of shared/utility services that are used to manager individual tool execution, these include Tool Manager, Scheduler, Accounts and so on

  2. Tools

    These components are a set of dedicated services which perform a specific business object, for example automated metric alerts and audits

# Metric Alerts

Definition: A Metric alert tool will raise an alert when certain predefined thresholds for a particular client resource are crossed.

Most Metric Alerts are configured as follows: for exceptions see source code

//using "generic" as placeholder for specific metric name
"genericAlertConfiguration": {
    "doCheck": true, //indicates if this check should be performed, if [false] then remaining properties on this object are ignored and check not performed
    "genericThreshold": 3000.00, //metric threshold 
    "negativeVarianceThreshold": 0.5, // negative variance threshold 
    "positiveVarianceThreshold": 0.5 // positive variance threshold 
},

# Terminology

  • period average metric value - the average value for a given metric over previous set of time windows - this is the baseline value for this metric

  • target metric value - the metric value for the time window being checked, typically this is the most recent given time window for which data is available

  • target variance - variance for target against baseline (target metric value / period average metric value) - 1

  • absolute target variance - the absolute value of target variance

# Conditions

An Alert will be raised when...

  1. The following is true:

    period average metric value OR target metric value GREATER THAN genericThreshold

  2. One of the following is true

    target variance LESS THAN zero(0) AND absolute target variance GREATER THAN negativeVarianceThreshold

    OR

    target variance GREATER THAN zero(0) AND absolute target variance GREATER THAN positiveVarianceThreshold
